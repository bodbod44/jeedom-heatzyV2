# jeedom-heatzy
Plugin heatzy pour la box domotique jeedom

Ce plugin permet de gerer les modules de la marque Heatzy

Il concerne les modules et radiateurs connectés Heatzy (Pilote_Soc, Pilote_Pro, Flam, Plugzy, Glow, Shine ...).
Les intercations avec l'application mobile officielle restent pleinement fonctionnelles.
Les actions faites sur jeedom sont visibles en temps réel sur l'application et vice versa
